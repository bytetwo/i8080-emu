# Intel 8080 Emulator
A primitive intel 8080 emulator written in C. I have used GTK+ as the windowing system and OpenGL to render the graphics.

## Dependencies
```
sudo apt-get install libgtk-3-dev
sudo apt-get install freeglut3-dev
```

## Build
```
make
```

## Run
```
./8080
```
